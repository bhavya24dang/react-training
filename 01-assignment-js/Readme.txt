Steps to run assignment.
1. Open index.html in Chrome
2. Enter the values of the array separated by ',' in input box. Example: 11, 12, 10, 11, 12, 13, 12
3. Click on button 'Check Frequency' will give you the max occurred number from the array with its frequency. 
# For above example, it will give: 12 ( 3 times )
